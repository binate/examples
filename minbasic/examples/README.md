# minbasic example programs

BASIC programs to run through minbasic's `cmd/run`. They are a little *meta* —
example programs for an example interpreter that itself lives in the Binate
examples repo.

```sh
./scripts/build-compiled.sh minbasic/cmd/run
out/minbasic/run minbasic/examples/hello.bas
# or interpreted:
./scripts/run-interpreted.sh minbasic/cmd/run minbasic/examples/mandelbrot.bas
```

`run.sh` here runs both programs in both modes and diffs them against
`expected/<name>.out`, so the programs (and the output below) can't silently
drift; it's picked up by the repo's e2e runner.

## `hello.bas`

The classic — a single `PRINT "HELLO, WORLD!"`.

Working within Minimal BASIC's limits is half the fun:

- **one statement per line** — every `LET` is its own line;
- **`IF cond THEN <line-number>`** only (no `THEN <statement>`), so the escape
  loop and the shading are `IF … THEN <line>` / `GOTO` chains;
- **variable names are a letter or letter+digit** — so `c` and `z` are `C`/`D`
  and `P`/`Q`, not `CR`/`ZI`;
- **no string functions** — each cell's glyph is chosen by an `IF` chain that
  `PRINT`s the right literal with a trailing `;` (which suppresses the newline);
  an empty `PRINT` ends the row.

The escape-time math is plain `float64` (`+ - * /` and a `> 4` test, no
`cast`/`INT`), so the picture is byte-identical compiled vs. interpreted.
